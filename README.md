# Desktop

Dotfiles for my desktop pc (13700K + 2070S).

## Software Setup

- distro: Arch Linux
- init system: systemd
- display manager: ly
- display server: X
- shell: fish
- window manager: bspwm (+ picom, sxhkd and polybar)
- terminal: kitty
- cursor: Capitaine light
- theme: Ultimate dark blue
- icons: Papirus dark
- font: Noto Sans (NF) / SourceCodePro
