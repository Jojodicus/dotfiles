# Laptop

Dotfiles for my laptop, a Lenovo IdeaPad 5 Pro (5800H + 3050).
These configs are not exactly pretty to look at, some settings are auto-generated.

## Software Setup

- distro: Arch Linux
- init system: systemd
- display manager: ly
- display server: X
- shell: bash
- window manager: bspwm (+ picom, sxhkd and polybar)
- terminal: kitty
- cursor: Bibata Ghost
- theme: Arc darkest
- icons: Papirus dark
- font: Noto Sans (Mono)
